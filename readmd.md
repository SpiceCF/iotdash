## BACKEND

STACK : GOLANG

- GOLANG :: Ex. API Sensor ( MOCK OR RANDOM )

- JWT ?
  | HEADER PAYLOAD SIGNATURE
  | {HASH BASE64}.{HASH BASE64}.{HASH BASE64}

  SYNMETRIC :: PASSWORD ( SAME KEY )
  ASYMMETRIC :: PUBLIC KEY / PRIVATE KEY ( DIFFERENT KEY )
